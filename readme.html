A guide to the Hybrid Original theme

Hybrid Original is a child theme of the Hybrid theme framework, which means it must be installed for this theme to work.

This theme is a view of the original Hybrid theme's style. Since Hybrid version 0.6 got a complete makeover, I wanted to release something for those people that are fond of the classic look.

Installation

  1. Make sure you've installed the Hybrid theme framework.
  2. Unzip the hybrid-original.zip file.
  3. Upload the hybrid-original folder to your /wp-content/themes directory.
  4. In your WordPress admin, go to Appearance > Themes.
  5. Activate Hybrid Original.
